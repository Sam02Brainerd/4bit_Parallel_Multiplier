# 🔹 4-bit Multiplier (Switch → LED Output)

## 📌 Description

This implementation verifies the functionality of a **4-bit parallel multiplier** using switches as input and LEDs as output.

---

## ⚙️ Working

* Inputs (A, B) are given via switches
* Multiplier computes product:

```
P = A × B
```

* Output displayed in **binary using LEDs**

---

## 🧪 Example

| A | B | Output (LEDs) |
| - | - | ------------- |
| 3 | 5 | 00001111      |
| 7 | 2 | 00001110      |

---

## 🚀 How to Run

1. Add design files
2. Add constraints file (LED + switches)
3. Set top module
4. Generate bitstream
5. Program FPGA

---

## 🎯 Purpose

* Basic verification of multiplier
* Understand FPGA I/O handling
